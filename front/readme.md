# Край Тур — фронтенд (хакатон)

Vite + React + TypeScript + Tailwind CSS. Только UI и клиентский флоу квиза, без бэкенда.

## Запуск

```bash
cd front
npm install
npm run dev
```

Сборка:

```bash
npm run build
npm run preview
```

## Маршруты

- `/` — лендинг (hero + блок «Как работает сервис»)
- `/quiz/1` … `/quiz/6` — шаги квиза
- `/quiz/done` — финальный экран с моковым итогом (ответы из Zustand)

## Ассеты

Референсные изображения экранов лежат в `src/assets/` и `src/assets/quiz/`. Иллюстрации справа на шагах квиза — кроп полноэкранных макетов (`object-position` + масштаб), отдельных вырезов карточек нет.

## Примечания по совпадению с макетом

- Шрифт: **Comfortaa** (Google Fonts) как ближайший к округлой геометрии из макета.
- Декоративные «петли» по краям квиза — упрощённые SVG, не вектор из Figma.
- Плавающие 3D-объекты на hero — упрощённые SVG-заглушки; в макете — отдельные 3D-рендеры.
- На шагах 3–6 в макете варианты ответов совпадают с блоком длительности — воспроизведено как на референсе.
